# Código de Conduta

Adotamos o [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Resumo

- Trate todos com respeito.
- Críticas técnicas sim; ataques pessoais não.
- Reportar violações: abrir issue privada ou contatar mantenedor (`mauricio.benjamin@reloverelations.com`).

## Aplicação

Mantenedores podem editar, remover, ou rejeitar comentários/PRs que violem este código. Casos graves: ban temporário ou permanente.
